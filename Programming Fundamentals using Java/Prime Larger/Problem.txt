Prime Larger

DESCRIPTION :

Write a Java program that accepts two integer values from the user and

1. prints the larger value if both the values are prime numbers. 
2. otherwise it prints smaller value and
3. it prints O if both the values are same.

Example:

Sample Input:

5
11

Expected Output:

11

Sample Input:

5
16

Expected output:

5

Sample Input:

5

Expected output:
